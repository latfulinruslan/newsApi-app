## About the project
- iOS client for [https://newsapi.org](https://newsapi.org)

## Technologies 
Project created using Swift, UIKit, Alamofire, Realm.

## Requirements
- iOS 13.0+ 
- Xcode 11 beta 6
- Swift 5

## Installation

```bash
$ git clone https://github.com/latfulinruslan/newsApi-app
```

